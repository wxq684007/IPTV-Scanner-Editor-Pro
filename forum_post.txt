[size=4][color=DarkOrange]从扫描到观看，全链路搞定[/color][/size]

[size=3][b]软件简介[/b][/size]

IPTV Scanner Editor Pro 是一款功能全面的 IPTV 工具，跨平台支持 Windows / macOS / Linux 桌面端与 Android 移动端。集成 MPV 播放引擎与 FFprobe 流探测，从频道扫描、批量验证、EPG 节目单、台标匹配到高清播放，一站式完成。

当前版本：[b]v48.2[/b] | 开源协议：MIT

[size=3][b][color=SeaGreen]核心亮点[/color][/b][/size]

[list=1]
[*][b]MPV 引擎驱动[/b] —— 基于 libmpv 的高性能流媒体播放，硬件解码（D3D11VA/NVDEC/VAAPI），HDR10/HDR10+/HLG/Dolby Vision 全支持，5 种 HDR 输出模式
[*][b]FCC 秒开换台[/b] —— IPTV 组播 FCC 代理，消除 IGMP 加入延迟，换台零黑屏
[*][b]EPG 电子节目单[/b] —— XMLTV/JSON 解析，智能匹配（tvg-id→tvg-name→频道名），日期导航，实时进度条，EPG 时间轴，节目提醒
[*][b]智能频道扫描[/b] —— IP 范围扫描，多协议（单播/组播/HTTP），FFprobe 探测，智能重试，批量自动分类/清理/匹配台标
[*][b]400+ 台标规则[/b] —— 内置央视、卫视、CGTN、咪咕等主流频道台标，在线加载 + 异步下载 + 本地缓存
[*][b]21 种主题组合[/b] —— 3 种颜色模式（自动/暗黑/日间）× 7 种视觉风格（拟态/扁平/拟物/毛玻璃/Win11/Mac/iOS），中英双语
[*][b]Android 原生端[/b] —— Kotlin Compose + MPV Native 渲染，TV/手机双模式，遥控器适配，画中画增强
[*][b]内置 Web 服务器[/b] —— aiohttp 轻量服务，频道列表/播放/EPG/扫描 RESTful API，8 Tab 管理后台，虚拟遥控器
[/list]
